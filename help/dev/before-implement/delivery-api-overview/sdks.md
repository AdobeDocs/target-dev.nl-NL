# SDK&#39;s

[!DNL Adobe Target] biedt [&#x200B; server-kant SDKs &#x200B;](../../implement/server-side/server-side-overview.md) aan om met [!DNL Target Delivery API] in wisselwerking te staan en de [!UICONTROL Target] integratie met andere [!DNL Adobe Experience Cloud] oplossingen zoals [!UICONTROL Adobe Analytics] en [!DNL Adobe Audience Manager] te verlichten.

Met de SDK&#39;s kunt u de beste praktijken volgen voor het beheren van sessies en cookies. Het wordt ten zeerste aanbevolen een SDK te gebruiken om complexiteiten weg te werken en onjuist gebruik van de API te voorkomen.

## Node.js SDK

Voor meer informatie, zie {het overzicht van SDK 0} Node.js [.](/help/dev/implement/server-side/node-js/overview.md)

[&#x200B; Node.js SDK &#x200B;](https://github.com/adobe/target-nodejs-sdk) is open-sourced en gehandhaafd binnen Github, waar u de documentatie en gebruiksinstructies kunt vinden.

Lees [&#x200B; changelog &#x200B;](https://github.com/adobe/target-nodejs-sdk/blob/main/CHANGELOG.md) voor Node.js SDK.
